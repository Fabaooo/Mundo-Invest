## 📋 Overview

Mundo Invest is a platform designed to support financial education and investment simulation through real-time market data, portfolio insights and AI-powered assistance.

The project focuses on helping users understand market behavior, simulate investment strategies and receive educational insights based on their investor profile.

## ✨ Key Features

- Real-time market data dashboard
- AI-powered financial assistant
- Portfolio simulation
- Risk profile-based insights
- Crypto and market asset monitoring
- Educational investment experience

## 🛠️ Tech Stack

**Frontend:** Next.js, React, TypeScript, Zustand, Recharts  
**Backend:** Node.js, Express.js, TypeScript  
**Database/Auth:** Supabase  
**AI:** Vercel AI SDK / OpenAI API  
**External APIs:** CoinGecko, Yahoo Finance  
**Other:** Redis, GitHub, REST APIs
